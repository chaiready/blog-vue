# 个人博客系统前端

> A Vue.js project for blog

## 技术栈

主要采用了：

1. Vue
2. axios
3. ElementUI
4. vue-echarts
5. vue-mdEditor
6. vue-router

还有其他一些琐碎的技术，就不在这里列举了。

## 快速运行



克隆项目到本地，进入根目录中运行下面的命令：

```shell
# 安装依赖
npm install

# 在 localhost:9090 启动项目
npm run dev
```
